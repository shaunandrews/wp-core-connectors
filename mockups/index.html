Connectors Mockups

Three design concepts for the new WordPress Core "Settings » Connectors" page. This centralized keychain will manage API keys, OAuth flows, and external service connections for WordPress 7.0.

Concept A: Card List

Inspired by Claude's connectors page

A clean, vertical list of connector cards with inline configuration.

  • Horizontal row layout with icons
  • Status badges and action buttons
  • Inline expansion for configuration
  • Overflow menus for advanced actions
View Concept A

Concept B: Grouped Fields

Inspired by AI Services plugin

Traditional WordPress settings with modern cards and clear grouping.

  • Grouped cards by service category
  • Traditional form fields with labels
  • Save Changes workflow
  • Status indicators and help links
View Concept B

Concept C: Grid + Detail Panel

Inspired by Untitled UI patterns

Modern, app-like interface with filtering and slide-out configuration.

  • Sidebar filtering by category
  • Card grid with toggle switches
  • Slide-in detail panels
  • Modern SaaS aesthetic
View Concept C

About This Project

These mockups explore different approaches for WordPress Core's new centralized connector management. The page will live between General and Writing in the Settings submenu, providing a unified interface for managing API keys for services like OpenAI, Anthropic, Google, and plugin-registered connections.
